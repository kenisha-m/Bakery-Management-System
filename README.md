A command-line based Bakery Management System built using Python and MySQL.  
This project manages bakery inventory, item details, pricing, and database storage through simple, efficient menu-driven operations.

#Features

- Add new bakery items with ID, name, quantity, and price  
- Update item details  
- Delete items from the database  
- View complete inventory  
- MySQL database integration  
- Automatic database and table creation  
- Error handling for invalid entries
